IDM 241 - Final: Micro Interaction Redesign for Grammarly

Found on www.grammarly.com

Current State

Thoughts

Currently, the micro interaction is very static. There is no transition/easing into the main hover state.

This micro interaction contains a child micro interaction when hovering over the icons that appear after the initial hover.

-

I would like to bring an eased transitional quality as well as improve the indicative/accessibility qualities using CSS.

This would take place through four levels of microinteraction.

Concept for redesign

1st lvl Interaction

Triggers

- On-Hover, Off-Hover.

Rules

On-Hover: Main Box

- First apply fog effect over box content and increase shadow intensity.

- Then move download and trash icons inwards from the sides.

- Finally fade in "Open".

Off-Hover: All

- Reverse the effects of on-hover.

Feedback (Visual)

On Hover

- Main Box, ease in shadow increase and fog effect.

- Download & Trash Icons, slide towards center from outside Main Box boundaries.

- Word "Open" fades in at center of box.

Off-Hover: All

- All effects play in reverse.

Loops & Modes

- None.

2nd lvl Interaction

Triggers

- On-Hover, Off-Hover.

Rules

On-Hover: Download & Trash Icons

- Change Download Icon fill to Grammarly Green, Trash Icon to red.

On-Hover: Rest of box

- Turn "Open" text green.

On-Hover: Red tab

- Display text bubble containing description text.

Off-Hover: All

- Reverse the effects of on-hover.

Feedback (Visual)

On Hover

- Download & Trash Icons, ease in green or red fill.

- "Open" text, ease in green color.

- Red tab, fade in text bubble.

Off-Hover: All

- All effects play in reverse.

Loops & Modes

- Mode, presence of Red Tab is based on external factor:

- If document contains issues.

3rd lvl Interaction

Trigger

- On-Click.

Rules

On-Click: Download Icon

- Launch default download procedures.

On-Click: Trash Icon

- Display trash confirmation modal.

Feedback (Visual)

On Click

- Trash Icon, fade in trash confirmation modal.

Loops & Modes

- None.

4th lvl Interaction

Triggers

- On-Click, On-Hover, Off-Hover

Rules

On-Hover: Cancel & Trash Buttons

- Change button fill to white.

- Change button text color to red ot blue.

On-Click: Cancel Button

- Close modal.

On-Click: Trash Button

- Delete respective box.

Feedback (Visual)

On Click

- Cancel Button, modal fades out.

- Trash Button, modal fades out, respective box disappears.

Loops & Modes

- None.
